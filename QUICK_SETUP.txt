## ‚öôÔ∏è TROUBLESHOOTING

**Error: "Multiple users on device"**
- Settings ‚Üí System ‚Üí Multiple users
- Remove all secondary users
- Try again

**"Permission denied" in Termux**
- Run: `su` first, then the dpm command

**"Device owner already set"**
- Run: `adb shell dpm remove-active-admin com.example.silent_installapp/.SilentInstallAdminReceiver`
- Then set it again
